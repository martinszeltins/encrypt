Encrypt text and files using AES-256 encryption.

# Encryption

### Encrypt text

```bash
npx @martinszeltins/encrypt "Your plain text here"
```

### Encrypt a file

```bash
npx @martinszeltins/encrypt myfile.txt
```

# Decryption

### Decrypt text

```bash
npx @martinszeltins/decrypt "Your plain text here"
```

### Decrypt a file

```bash
npx @martinszeltins/decrypt myfile.txt
```
